# 🎓 ABC College Student Portal

A modern, fully responsive, frontend-only college management portal built using React, Vite, TypeScript, and TailwindCSS.

## 🚀 Live Demo

🔗 https://sole-sneaks.vercel.app

---

# 📌 Features

## 🔐 Authentication
- Role-based login system
- Student / Faculty / Admin access
- Remember Me functionality
- Session token support
- Forgot password with OTP verification
- Password reset flow

---

## 👨‍🎓 Student Portal
- Academic dashboard
- Attendance tracking
- Marks & CGPA management
- Fee payment system
- Timetable & exam schedules
- Assignment submissions
- Library management
- Notifications center
- Digital ID card
- PDF & PNG downloads

---

## 👨‍🏫 Faculty Portal
- Faculty dashboard
- Subject assignment tracking
- Student attendance marking
- Assignment creation
- LMS material uploads
- Timetable management

---

## 👨‍💼 Admin Portal
- Leave approval/rejection
- User management
- Attendance monitoring
- Fee monitoring
- Notification management

# ⚡ Installation

## Clone Repository

```bash
git clone https://github.com/Kushal-ctrl-del/sole.git
```

## Navigate to Project

```bash
cd sole
```

## Install Dependencies

```bash
pnpm install
```

## Start Development Server

```bash
pnpm dev
```

---

# 🏗 Build Project

```bash
pnpm build
```

- localStorage
- sessionStorage

No backend server is used.
